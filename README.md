# Claude Camp W2 练习说明
本目录包含 4 个 Python 小程序：学员花名册管理器，待办事项清单，文本词频统计器和计算器。以下说明各自功能、运行方式、运行环境和使用方法。

## 1. 学院花名册管理器
### 功能介绍
| 功能 | 说明 |
|------|------|
| 添加学员 | 输入姓名、邮箱、加入日期，将学员信息存入名册 |
| 删除学员 | 根据姓名查找并移除对应学员信息 |
| 查询学员 | 根据姓名搜索并显示学员的详细信息 |
| 友好提示 | 输入无效选项或查找不存在的学员时，程序不会崩溃，会给出提示 |

### 运行环境
- Python 3.6 及以上
- 无需安装任何第三方库，使用 Python 内置功能即可运行

### 使用方法
1.运行程序
```bash
cd claude-camp-w2
学员花名册管理器.py
```
2. 根据菜单提示选择功能
