# Claude Camp W2 练习说明
本目录包含 4 个 Python 小程序：学员花名册管理器，待办事项清单，文本词频统计器和计算器。以下说明各自功能、运行方式、运行环境和使用方法。

## 1. 学院花名册管理器
### 功能介绍
| 功能 | 说明 |
|------|------|
| 添加学员 | 输入姓名、邮箱、加入日期，将学员信息存入名册 |
| 删除学员 | 根据姓名查找并移除对应学员信息 |
| 查询学员 | 根据姓名搜索并显示学员的详细信息 |
| 友好提示 | 输入无效选项或查找不存在的学员时，程序不会崩溃，会给出提示 |

### 运行环境
- Python 3.6 及以上
- 无需安装任何第三方库，使用 Python 内置功能即可运行

### 使用方法
1.运行程序
```bash
学员花名册管理器.py
```
2. 根据菜单提示选择功能

# 2. 文本词频统计器
## 功能介绍
| 功能 | 说明 |
|------|------|
| 输入文本 | 在窗口中输入英文文本 |
| 词频统计 | 统计每个单词出现的次数，存储为字典 |
| 排序输出 | 按出现次数从高到低排序展示结果 |
| 友好提示 | 格式化输出，单词与次数对齐显示 |

## 运行环境
- Python 3.6 及以上
- 无需安装任何第三方库，使用 Python 内置功能即可运行

## 使用方法
1. 运行程序
```bash
python 词频统计器.py
```
2. 根据提示输入一段英文文字
3. 程序将自动输出词频统计结果，示例如下：

```
词频统计结果如下，已按出现次数从高到低排序
-------------------------
in              3
magnificent     2
mind            1
```

# 3. 待办事项清单
## 功能介绍
| 功能 | 说明 |
|------|------|
| 添加事项 | 输入待办内容，将事项存入清单 |
| 删除事项 | 根据事项内容查找并移除已完成事项 |
| 查看清单 | 显示当前所有待办事项 |
| 本地保存 | 自动将清单保存至本地 todos.json 文件 |
| 数据加载 | 程序启动时自动加载上次保存的数据 |
| 友好提示 | 输入无效选项或删除不存在的事项时，程序不会崩溃，会给出提示 |

## 运行环境
- Python 3.6 及以上
- 无需安装任何第三方库，使用 Python 内置功能即可运行

## 使用方法
1. 运行程序
```bash
python 待办事项清单.py
```
2. 根据菜单提示选择功能

# 4. 简易计算器
## 功能介绍
| 功能 | 说明 |
|------|------|
| 加法 | 输入两个数字和运算符 +，返回相加结果 |
| 减法 | 输入两个数字和运算符 -，返回相减结果 |
| 乘法 | 输入两个数字和运算符 *，返回相乘结果 |
| 除法 | 输入两个数字和运算符 /，返回相除结果 |
| 防崩溃 | 输入非数字或除数为 0 时，程序不会崩溃，会给出提示 |
| 退出 | 任意步骤输入 q，程序优雅退出 |

## 运行环境
- Python 3.6 及以上
- 无需安装任何第三方库，使用 Python 内置功能即可运行

## 使用方法
1. 运行程序
```bash
python 计算器.py
```
2. 根据提示依次输入第一个数字、运算符、第二个数字
3. 任意步骤输入 `q` 可随时退出
